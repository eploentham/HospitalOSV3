6.4 ตัวอย่างการออกรายงาน 11รง5
6.4.1 ตัวอย่างการออกรายงาน 11รง5 ส่วนที่ 1
          
รูปที่ 35 แสดงการออกรายงาน 11 รง5 ส่วนที่ 1 (OPD/PCU )
 
รูปที่ 36 แสดงการบันทึกรายงาน 11 รง5 ส่วนที่ 1
 
รูปที่ 37 แสดงผลการบันทึกรายงาน 11 รง5 ส่วนที่ 1
  ขั้นตอนการออกรายงาน 11 รง5 ส่วนที่ 1
      1. เลือกรายงาน 11รง5 ส่วนที่ 1 จะแสดงรูปแบบของรายงานที่ต้องการออกรายงาน
  2. เมื่อเลือกรายงาน 11รง5 ส่วนที่ 1 รายงานส่วนนี้จะ Default ไว้ที่ OPD/PCU เสมอ และ แสดงข้อความแจ้งให้ผู้ใช้งานทราบถึงรายละเอียด ของการออกรายงานส่วนที่ 1
  3. เลือกวันที่ เริ่มต้น ที่ต้องการออกรายงาน
  4. เลือกวันที่ สิ้นสุด ที่ต้องการออกรายงาน
  5. กดปุ่ม Query สำหรับการดึงรายงาน 11 รง5 ส่วนที่ 1 (OPD/PCU )
  6. แสดงรายละเอียดของรายงาน 11 รง5 ส่วนที่ 1 (OPD/PCU ) หลังจากกดปุ่ม Query
  7. เลือกชนิดของไฟล์ ที่ต้องการบันทึก ( Text ไฟล์ หรือ Excel ไฟล์)
  8. กดปุ่ม บันทึก สำหรับการบันทึกรายงาน 11รง5 ส่วนที่1 ( OPD/PCU,IPD,Refer)
  9. ปุ่ม บันทึก โปรแกรมจะแสดงหน้าต่างสำหรับเลือก Folder สำหรับเก็บไฟล์รายงาน
  10. เลือก Directory ที่มี Folder อยู่ หรือที่ต้องการจะจัดเก็บ
  11. เลือก Folder ที่ต้องการจะจัดเก็บ
  12. แสดง Path ที่เก็บไฟล์รายงาน
  13. จะได้ไฟล์รายงานที่ต้องการ
