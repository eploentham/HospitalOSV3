6.3 ตัวอย่างการออกรายงาน 506
6.3.3 ตัวอย่างการออกรายงาน 506 ข้อมูลติดตามผู้ป่วย 506
          
รูปที่ 32 แสดงการออกรายงาน ข้อมูลติดตามผู้ป่วย 506
 
รูปที่ 33 แสดงการบันทึกรายงาน
 
รูปที่ 34 แสดงผลการบันทึกรายงาน ข้อมูลติดตามผู้ป่วย 506
  ขั้นตอนการออกรายงาน 506 ข้อมูลติดตามผู้ป่วย 506
      1. เลือกรูปแบบรายงานเป็น รายงาน 506 ข้อมูลติดตามผู้ป่วย 506
  2. เลือกวันที่เริ่มต้นสำหรับการค้นหา(กรณีนี้เลือก 01/04/2548 )
  3. เลือกวันที่สิ้นสุดสำหรับการค้นหา (กรณีนี้เลือก 30/04/2548 )
  4. เลือกสถานะการจำหน่าย ที่ต้องการค้นหา ( กรณีนี้ เลือกสถานะ ทั้งหมด)
  5. กดปุ่ม Query โปรแกรมจะทำการค้นหาข้อมูลตามเงื่อนไขของวันที่ ที่ระบุข้างต้น
  6. ข้อมูลที่ได้จากการ query จะถูกแสดงในตาราง รายละเอียดของรายงาน
  7. ถ้าต้องการ บันทึกรายงานออกเป็นไฟล์ ให้เลือกชนิดของไฟล์ ปกติจะเป็น Text ไฟล์ (กรณีนี้เลือกเป็นxls ไฟล์)
  8. กดปุ่ม บันทึก โปรแกรมจะแสดงหน้าต่างสำหรับเลือก Folder สำหรับเก็บไฟล์รายงาน
  9. เลือก Directory ที่มี Folder อยู่ หรือที่ต้องการจะจัดเก็บ
  10. เลือก Folder ที่ต้องการจะจัดเก็บ
  11. แสดง Path ที่เก็บไฟล์รายงาน
  12. กดปุ่ม Open เพื่อระบุที่เก็บและให้โปรแกรมทำการเขียนไฟล์ไปเก็บยัง Folder ที่ระบุข้างต้น
