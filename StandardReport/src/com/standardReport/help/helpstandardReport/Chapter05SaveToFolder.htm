5. การเลือก Folder ที่ต้องการจะจัดเก็บไฟล์
                สำหรับ การเลือก Folder ที่ต้องการจัดเก็บไฟล์ จะเป็นขั้นสุดท้ายของการเลือกรายละเอียด จะต้องกดปุ่มบันทึก โปรแกรมถึงจะให้เลือก Folder ที่ต้องการจะจัดเก็บ ดังรูปที่ 12 และ รูปที่ 13
          
รูปที่ 12 การกดปุ่ม บันทึก เพื่อเลือก Folder ที่เก็บไฟล์
 
รูปที่ 13 แสดง หน้ากำหนด Folder ที่เก็บ ไฟล์
     

                เมื่อเลือก Folder ที่เก็บไฟล์ ได้แล้ว ให้กดปุ่ม Open โปรแกรมจะทำการ เขียนข้อมูลที่ Query ไว้แล้วลงเก็บในไฟล์ และจะเก็บไฟล์ไว้ตามตำแหน่งที่เลือกไว้
