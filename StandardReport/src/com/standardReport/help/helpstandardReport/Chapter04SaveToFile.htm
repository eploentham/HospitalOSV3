4. การเลือกรูปแบบของไฟล์ที่ต้องการจะบันทึก

                การเลือกรูปแบบของไฟล์ที่ต้องการเพื่อต้องการให้บันทึกข้อมูลที่ query เสร็จแล้วอยู่ในรูปแบบของไฟล์ประเภทไหน จะมีด้วยกัน 2 ประเภท คือ
  1. xls ไฟล์ สามารถเปิดได้กับ excel
  2. text ไฟล์ สามารถเปิดได้กับ notepad หรือโปรแกรมอื่นๆที่สามารถเปิด text ไฟล์ได้สำหรับการออกรายงานมาตรฐาน จะกำหนดอยู่ที่ text ไฟล์
          
รูปที่ 11 แสดง ชนิดของไฟล์ ที่ต้องการจะให้บันทึก
