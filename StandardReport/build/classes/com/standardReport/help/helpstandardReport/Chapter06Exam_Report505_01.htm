6.2 ตัวอย่างการออกรายงาน 505
6.2.1 ตัวอย่างการออกรายงาน 505 รายกลุ่ม
          
รูปที่ 17 แสดงการออกรายงาน 505 รายกลุ่ม
 
รูปที่ 18 แสดงการบันทึกรายงาน
 
รูปที่ 19 แสดงผลการบันทึก รายงาน 505 รายกลุ่ม
  ขั้นตอนการออกรายงาน 505 รายกลุ่ม
      1. เลือกรูปแบบรายงานเป็น รายงาน 505 รายกลุ่ม
  2. เลือกวันที่เริ่มต้นสำหรับการค้นหา(กรณีนี้เลือก 01/08/2548 )
  3. เลือกวันที่สิ้นสุดสำหรับการค้นหา (กรณีนี้เลือก 31/08/2548 )
  4. กดปุ่ม Query โปรแกรมจะทำการค้นหาข้อมูลตามเงื่อนไขของวันที่ ที่ระบุข้างต้น
  5. ข้อมูลที่ได้จากการ query จะถูกแสดงในตาราง รายละเอียดของรายงาน
  6. ถ้าต้องการ บันทึกรายงานออกเป็นไฟล์ ให้เลือกชนิดของไฟล์ ปกติจะเป็น Text ไฟล์ (กรณีนี้เลือกเป็นText ไฟล์)
  7. กดปุ่ม บันทึก โปรแกรมจะแสดงหน้าต่างสำหรับเลือก Folder สำหรับเก็บไฟล์รายงาน
  8. เลือก Directory ที่มี Folder อยู่ หรือที่ต้องการจะจัดเก็บ
  9. เลือก Folder ที่ต้องการจะจัดเก็บ
  10. แสดง Path ที่เก็บไฟล์รายงาน
  11. กดปุ่ม Open เพื่อระบุที่เก็บและให้โปรแกรมทำการเขียนไฟล์ไปเก็บยัง Folder ที่ระบุข้างต้น
