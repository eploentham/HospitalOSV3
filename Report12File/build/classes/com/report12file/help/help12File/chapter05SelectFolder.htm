5. การเลือก Folder ที่ต้องการจะจัดเก็บไฟล์

                สำหรับ การเลือก Folder ที่ต้องการจัดเก็บไฟล์ จะเป็นขั้นสุดท้ายของการเลือกรายละเอียดจากแล้ว จะต้องกดปุ่มตกลง โปรแกรมถึงจะให้เลือก Folder ที่ต้องการจะจัดเก็บ ดังรูปที่ 12 และ รูปที่ 13

          

รูปที่ 12 การกดปุ่ม ตกลงเพื่อทำการ query และ เลือก Folder ที่เก็บไฟล์
          

รูปที่ 13 แสดง หน้ากำหนด Folder ที่เก็บ ไฟล์

                เมื่อเลือก Folder ที่เก็บไฟล์ ได้แล้ว ให้กดปุ่ม Open โปรแกรมจะทำการ Query ข้อมูล และจะเก็บไฟล์ไว้ตามตำแหน่งที่เลือกไว้
