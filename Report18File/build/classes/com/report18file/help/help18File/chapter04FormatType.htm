4. การเลือกรูปแบบของไฟล์ที่ต้องการจะบันทึก

                การเลือกรูปแบบของไฟล์ที่ต้องการเพื่อต้องการให้บันทึกข้อมูลที่ query เสร็จแล้วอยู่ในรูปแบบของไฟล์ประเภทไหน จะมีด้วยกัน 3 ประเภท คือ

1.       dbf ไฟล์ สามารถเปิดได้กับ foxpro

2.       xls ไฟล์ สามารถเปิดได้กับ excel

3.       text ไฟล์ สามารถเปิดได้กับ notepad หรือโปรแกรมอื่นๆที่สามารถเกิด text ไฟล์ได้

สำหรับการออกรายงานของ 18 แฟ้ม จะกำหนดอยู่ที่ text ไฟล์

          

รูปที่ 11 แสดง ชนิดของไฟล์ ที่ต้องการจะให้บันทึก
