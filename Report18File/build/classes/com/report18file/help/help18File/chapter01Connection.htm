1. การติดต่อฐานข้อมูล

                สำหรับ การทำงานในครั้งแรกของโปรแกรม เมื่อปิดโปรแกรมขึ้นมา โปรแกรมจะแสดง Dialog ดังรูปที่ 1 นั้นหมายความว่าโปรแกรมยังไม่ได้เชื่อมต่อกับฐานข้อมูล ให้ผู้ใช้ทำการเชื่อมต่อกับฐานข้อมูลด้วย ดัง รูปที่ 2

          

รูปที่ 1 แสดงข้อความครั้งแรกที่เปิดโปรแกรมถ้ายังไม่เชื่อมต่อกับฐานข้อมูล

          

รูปที่ 2 ผู้ใช้ต้องกรอกรายละเอียดเพื่อทำการเชื่อมต่อกับฐานข้อมูล

คำอธิบาย

เครื่องแม่ข่าย  เป็น หมายเลข IP ของเครื่อง Server ที่มีฐานข้อมูลอยู่ (เช่น 192.168.1.1)

ฐานข้อมูล เป็นฐานข้อมูลที่ต้องการใช้งาน (เช่น hospital_os)

พอร์ต เป็น พอร์ต ที่ใช้ในการติดต่อ โดยปกติจะเป็น 5432

ชื่อผู้ใช้ฐานข้อมูล เป็น ชื่อของผู้มีสิทธิเข้าใช้งานฐานข้อมูล

รหัสผ่าน เป็น รหัสผ่านของผู้มีสิทธิเข้าใช้งานฐานข้อมูล

ขั้นตอนการทำงาน

1.       กรอกรายละเอียดที่แสดงในรูปที่ 2 ให้ครบ

2.       กดปุ่ม ตกลง

3.       กรณีที่ พิมพ์ เครื่องแม่ข่าย, ฐานข้อมูล,พอร์ต,ชื่อผู้ใช้ฐานข้อมูล หรือ รหัสผ่าน ผิดโปรแกรมจะแสดง Dialog ดังรูปที่ 3

รูปที่ 3 กรอกรายละเอียดผิดพลาด

4.       เมื่อกรอกถูกต้องและ กดปุ่มตกลงแล้ว โปรแกรมจะแสดง ผลดังรูปที่ 4 พร้อมใช้งาน

          

รูปที่ 4 โปรแกรมพร้อมใช้งาน
