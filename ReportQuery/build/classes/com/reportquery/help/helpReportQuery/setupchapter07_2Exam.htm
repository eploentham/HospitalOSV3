การกรอกข้อมูล

 

รูปที่ 23 แสดง การกรอกข้อมูล

1. กรอกรายละเอียดข้อมูล

1         รหัส ใช้ในการกำหนดว่า คำสั่ง SQL นี้มีรหัสลำดับที่เท่าไร

2.       คำอธิบาย ให้ทราบว่า คำสั่ง SQL นี้ ใช้ทำอะไรหรือสื่อถึงอะไร

3.       ต้องการค้นหาด้วยวันที่  ระบุประเภทของ SQL (ไม่ต้องทำเครื่องหมายถูก)

4.       คำสั่ง SQL กรอกคำสั่ง SQL ที่ต้องการ

5.       Active กำหนดให้แสดงในหน้า Query รายงานหรือไม่

 

การบันทึก

 รูปที่ 24 กดบันทึกข้อมูล

1.       กดปุ่มบันทึก เมื่อกรอกรายละเอียดเสร็จ

2.       โปรแกรมจะแสดงรายการให้ทางซ้ายมือ

กรณีที่ มีการกรอกข้อมูล ที่ซ้ำกับที่มีอยู่ แล้วโปรแกรมจะแสดง Dialog ดังรูป ที่ 25 และ 26

รูปที่ 25 แสดงข้อความเมื่อกรอกรหัสซ้ำ จะแสดงเมื่อมีการกดปุ่มบันทึก

          

รูปที่ 26 แสดงข้อความเมื่อกรอกคำอธิบายซ้ำ จะแสดงเมื่อมีการกดปุ่มบันทึก

และเมื่อกรอก คำสั่ง SQL ผิดรูปแบบ จะแสดงดังรูปที่ 27

          

          

รูปที่ 27 แสดงข้อความเมื่อ คำสั่ง SQL ผิดรูปแบบ จะแสดงเมื่อมีการกดปุ่มบันทึก
