3. การเลือกรายการ SQL ที่ต้องการ

                เมื่อทำการค้นหารายการที่มีอยู่และแสดงรายการให้ผู้ใช้แล้ว และต้องการจะดูรายละเอียดหรือคำสั่ง SQL สามารถดูได้โดยการเลือก รายการที่ต้องการ ดังรูปที่ 8 หาก ทำการค้นหา แล้วไม่พบรายการอะไรเลย จะต้องทำการเพิ่มรายการ(ดูหัวข้อ 4 การเพิ่ม และลบรายการ)

          

รูปที่ 8 แสดง การเลือกรายการที่ต้องการเพื่อให้แสดงผลทางด้วยขวามือ
