2. การค้นหารายการ SQL

                สำหรับการค้นหา รายการ SQL หรือรายงาน SQL นั้นสามารถค้นหาได้โดยใช้ รหัส หรือ คำอธิบาย รูปแบบที่จะใช้ในการค้นหาจะเป็น ประกอบด้วย ตัวอย่างของการค้นหา ดังรูปที่ 5 เมื่อ กรอกคำที่จะค้นหา เสร็จแล้ว สามารถกด Enter หรือ กดปุ่ม ค้นหาก็ได้ โปรแกรมจะทำการค้นหา และนำมาแสดงให้ดังรูปที่ 5 แต่หากโปรแกรมค้นหาแล้วไม่เจอคำที่ผู้ใช้ต้องการโปรแกรมจะไม่แสดง ข้อมูลให้ ดังรูปที่ 6 นอกจากนั้น ผู้ใช้สามารถที่จะค้นหา ข้อมูลรายการ SQL ที่ไม่แสดงได้โดยการ นำเครื่องหมายถูก หน้า Active ออก (อยู่ใต้ค้นหา)โดยการคลิกที่เครื่องหมายถูก และกดปุ่มค้นหา โปรแกรมจะทำการค้นหารายการ SQL ที่ไม่แสดงให้ ดังรูปที่ 7

          

รูปที่ 5 แสดงการค้นหา และการแสดงผลจากการค้นหา

          

รูปที่ 6 แสดงการค้นหา และแสดงผลที่ค้นหาไม่เจอ

          

รูปที่ 7 แสดงการค้นหารายการที่ ไม่แสดง โดยการเอาเครื่องหมายถูกหน้า Active ออก
