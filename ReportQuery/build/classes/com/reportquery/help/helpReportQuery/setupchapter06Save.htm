6. การบันทึก รายการคำสั่ง SQL

                ในการบันทึก รายการคำสั่ง SQL นั้นผู้ใช้จะต้องกรอกข้อมูลให้ครบ และจะต้องไม่ซ้ำกันที่มีอยู่ โดยเฉพาะ รหัส และ คำอธิบาย  และสามารถใช้ได้กับการแก้ไขรายการคำสั่ง SQL ด้วย โดยการกดปุ่มบันทึก ดังรูปที่ 12  สามารถบันทึกถ้าหากต้องการให้รายการคำสั่ง SQL นั้นไม่แสดงในหน้า Query รายงาน จะต้องเอาเครื่องหมายถูกหน้า Active (อยู่ด้านล่าง)ออก และ กดปุ่มบันทึก ดังรูปที่ 13

          

รูปที่ 12 แสดงการบันทึกข้อมูล เมื่อ กรอกข้อมูล หรือแก้ไขข้อมูลเสร็จเรียบร้อยแล้ว

          

รูปที่ 13 แสดงการ กำหนดให้รายงาน SQL นี้ ไม่ต้องไปแสดงในหน้า Query รายงาน

                เมื่อกดบันทึกแล้ว โปรแกรมจะทำการตรวจสอบ Syntax ของ SQL ให้ว่าตรงตามประเภทของการรายงานหรือไม่ เช่น ถ้า เลือก ต้องการค้นหาด้วยวันที่ แต่ไม่มีเครื่องหมาย ? เลย โปรแกรมจะขึ้นมาเตือนผู้ใช้ เป็นต้น
